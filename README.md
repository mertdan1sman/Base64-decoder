🔓 Base64 Decoder/Encoder
Bu proje, Base64 formatında kodlanmış metinleri çözmek ve metinleri Base64 formatına dönüştürmek için hazırlanmış basit, hızlı ve etkili bir Python aracıdır.

🚀 Özellikler
Hızlı Çözme: Herhangi bir Base64 dizisini saniyeler içinde okunabilir metne dönüştürür.

Hata Yönetimi: Geçersiz karakterler veya bozuk formatlar için hata yakalama mekanizmasına sahiptir.

Kullanıcı Dostu: Karmaşık kütüphaneler gerektirmez, Python'un standart paketlerini kullanır.

🛠️ Kurulum
Bu aracı çalıştırmak için bilgisayarınızda Python yüklü olması yeterlidir. Ekstra bir kurulum gerekmez.
